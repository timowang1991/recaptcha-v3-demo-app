# How to run

* create `.env` file with the following content (the key comes from [Google reCAPTCHA admin site](https://www.google.com/u/1/recaptcha/admin))
```
RECAPTCHA_SITE_KEY=<server_side_key>
```

* run
```
docker-compose up
```

* open browser at http://localhost:3000
